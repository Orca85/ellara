Formelblad – Ellära

Alla formler på ett ställe

1. Ohms lag

U = R · I
R = UI
I = UR
Gäller för DC och resistiva AC-kretsar

2. Effektlagen

P = U · I
P = I² · R
P = U²R
P i Watt (W)

3. Resistans i ledare

R = ρ · LA
ρCu = 0,018 Ω·mm²/m
ρAl = 0,028 Ω·mm²/m
L i meter, A i mm²

4. Spänningsfall

Ufall = 2 · ρ · L · IA
Max tillåtet: 4 % av Unom
Vid 230 V: max 9,2 V
Faktorn 2 = fram- och återledare

5. Kopplingslagar

Seriekoppling

Rtot = R₁ + R₂ + R₃ + …
I är lika överallt
Utot = U₁ + U₂ + …

Parallellkoppling

1Rtot = 1R₁ + 1R₂ + …
U är lika överallt
Itot = I₁ + I₂ + …

Kirchhoffs spänningslag

ΣU = 0

Kirchhoffs strömlag

ΣIin = ΣIut

6. Transformatorn

U₁U₂ = N₁N₂
U₁ · I₁ = U₂ · I₂
Fler varv → högre spänning. Effekten bevaras.

7. Trefas & AC

Spänning

Ulinje = Ufas · √3 ≈ 230 · 1,732 ≈ 400 V

Frekvens

f = 1T    T = 1f    ω = 2 · π · f

Reaktans

XL = 2·π·f·L = ω·L
XC = 12·π·f·C = 1ω·C

Impedans

Z = √(R² + X²)    U = Z · I

Effektivvärde

Ueff = Umax√2    Ieff = Imax√2

Resonansfrekvens

f0 = 12·π·√(L·C)

Trefaseffekt

Ptot = 3 · Ufas · I = √3 · Ulinje · I

8. Effekttyper (P, Q, S)

S = √(P² + Q²)
cos φ = PS
P = U · I · cos φ
P = aktiv effekt (W) — nyttigt arbete
Q = reaktiv effekt (VAr) — pendlar i spolar/kondensatorer
S = skenbar effekt (VA) — vad elnätet levererar

9. Y- och D-koppling

Y-koppling (stjärna)

Ulinje = √3 · Ufas
Ilinje = Ifas
Neutral­ledare möjlig

D-koppling (triangel)

Ulinje = Ufas
Ilinje = √3 · Ifas
Ingen neutral­ledare
Y/D-start: Motor startar i Y (1/3 av D-effekt), kopplas om till D vid normalt varvtal.

10. Verkningsgrad

η = PutPin · 100 %
Put = η · Pin
Pförlust = Pin − Put
η = 1 (100 %) = ideal, ingen förlust. Typiskt 0,8–0,98 för elmotorer.

11. Kondensatorer & Induktorer

Kondensatorer — serie

1Ctot = 1C₁ + 1C₂ + …

Kondensatorer — parallell

Ctot = C₁ + C₂ + …

Induktorer — serie

Ltot = L₁ + L₂ + …

Induktorer — parallell

1Ltot = 1L₁ + 1L₂ + …
C: omvända regler jämfört med R. L: samma regler som R.

12. Batterier

Kapacitet

Ah = A · h
A = Ahh
h = AhA

Inre resistans (Rᵢ)

Rtot = R + Ri
Ulast = E − I · Ri
I = ER + Ri

Koppling av batterier

Serie: U adderas, Ah samma
Parallell: U samma, Ah adderas
E = EMK (tomgångsspänning). Klämspänning sjunker under belastning pga. Ri.
